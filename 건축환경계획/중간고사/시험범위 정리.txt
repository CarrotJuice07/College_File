물리학적 정의 : 인간에 시각에 의해 지각될 수 있는 가시광선 영역대의 파장을 갖는 전자기파의 복사에너지
빛의 감지를 위한 요소 : 소스, 물체, 눈, 뇌

용어 정리
광도 : 어떤 방향의 광원의 밝기 			(단위 : cd 칸델라)
휘도 : 어떤 방향의 광원면의 단위 면적당 광도		(단위 : cd/m^2)
광속 : 빛의 양					(단위 : lm 루멘)
조도 : 단위 면적에 입사하는 광속			(단위 : lx 럭스)
투과율, 반사율 : 물체에 비쳤을 때 통과되는 비율, 반사되는 비율

실내 반사율 권장값 : 천장(80%~90%) > 벽(40%~60%) > 가구(25%~45%) > 바닥(20%~40%)


조도
자연광은 태양은 100000lx, 맑은 하늘일 때 태양빛을 받은 면이 10000lx정도 나옴
인공광은 사무실이 400~500lx, 공장이나 체육관이 200~300lx정도 나옴
1. 거리의 역자승법칙
광원의 빛의 양이 일정할 때 점광원으로부터 거리가 멀수록 빛이 비추는 면적은 거리의 제곱에 비례해 늘어남
즉 조도(E)는 광도(I)에 비례하고, 거리(R)제곱에 반비례함
공식 : E = I/R^2 (lx)
2. 입사각 코사인법칙, 입사각 여현의 법칙 (Lambert의 코사인법칙)
빛이 경사가 있는 표면에 입사할 때 조도는 각 코사인에 비례
공식 : E'(경사면조도) = E(수평면조도) X cos@(입사각)

균제도 : 조도에 의한 빛의 균일한 정도를 나타내는 것
균제도 = 최저조도/최고조도
균제도가 1에 가까울수록 조도분포가 균일함을 나타내는 것이다.
창은 높게>낮게, 직접조명>간접조명, 조명의 간격은 촘촘>넓게, 채광위치는 북측>남측으로 하는 것이 변화가 크기 않아 균제도가 높음

휘도 : 어떤 방향으로 반사하는 빛의 세기, 보는 방향에 따라 차이가 발생
휘도 분포는 작업에 있어서 시각적 쾌적감에 영향을 미침
휘도레벨은 표면반사율과 조도레벨에 의해 좌우됨
휘도비 : 작업면의 휘도, 작업면or광원주위의 휘도 간의 비
이상적인 작업면 휘도는 주변 휘도보다 약간 큰 것이 좋다.

휘도와 눈부심(glare)
시야 범위에 휘도가 높은 부분이 있으면 눈부심을 느껴서 눈의 피로나 불쾌감을 느낌
눈부심은 광원의 휘도와 입체각의 영향을 받음
눈부심은 불편함을 주는 요소이기 때문에 창의 디자인과 주광의 디자인시 중요한 요소로 고려됨

직접글레어 : 휘도가 높은 물체가 집접 시야에 노출될 때 발생
간접글레어(반사글레어) : 휘도가 높은 물체가 직접 보이진 않지만 반사광, 반사면에 의해 발생

광막반사 : 정반사나 호나산반사에 의해 빛이 중첩되어 대상물에 빛의 막이 덮힌듯 보이는 현상

색온도 : 광원이 방출하는 빛의 색을 물리적인 척도로 나타낸 것
색온도가 작아질수록 적색 -> 주황색 -> 백색 -> 청백색 -> 청색 순서로 나타남
색온도의 단위는 K(켈빈) 임

연색성 : 물체색이 보이는 성질을 나타내는 광원의 특성
단위는 연색성지수를 의미하는 CRI 혹은 Ra로 표시됨, 최대값은 Ra 100
Ra85 이상은 연색성이 좋다고 말함


시지각이론
명순응 : 밝기에 익숙하게 되는 작용
암순응 : 어둠에 익숙하게 되는 작용
시감도 : 빛의 색에 따라 느끼는 밝기의 정도

퍼킨제효과 : 빛의 파장에 따라 눈의 색채 감광도가 면화하는 현상

시효율 : 얼마나 효율적으로 시지각작업이 이루어지는가

주광(Daylight)의 구성 : 직사일광, 황산천공광, 반사광
전천공조도 : 직사일광 + 천공광의 조도
설계용 천공조도 : 표준담천공일 때의 천공조도


천공
천공비율 : 천공 수평 일사랑 / 전천공 수평 일사량
청천공 : 천공비율 < 0.3
부분 담천공 : 0.3 < 천공비율 < 0.8
담천공 : 천공비율 > 0.8

천공커버 : 천공의 구름양에 의해 천공을 구분하는 것
청천공 : 0.0 ~ 0.3
부분 담천공 : 0.4 ~ 0.7
담천공 : 0.8 ~ 1.0

표준 담천공 : 북유럽 국가들에 나타나는 천공조건 (CIE 담천공)
천정과 수평면의 휘도비가 약 3:1
보통 인공천공돔을 통해 표준담천공 환경을 구현하여 측정

주광률(DF) : 실내 밝기를 나타내는 지표 (%)
표준담천공 조건에서 외부천정고도에 대한 실내조도의 비
주광률(DF) = 천공성분(SC) + 실외반사성분(ERC) + 실내반사성분(IRC)

채광주간 : 태양의 고도가 10도 이상인 천공이 채광에 유효하다고 보고, 그 시간대를 의미함

자연채광성능 평가방법 : 컴퓨터 시뮬레이션, 축소모형, 실물대모형 실험, 현장측정
물리량 평가(각종 수치 측정), 심리적 평가(주관적 감성), 생리학적 평가(몸의 반응)


자연 채광률 설계시 유의사항
1) 직사일광은 실내로 직접 입사되지 않도록 함
2) 직사일광의 이용은 두 업무공간이 아닌 곳에서 하도록 함
3) 반사되어 유입되는 빛을 이용할것
4) 강한 직사일광을 조절하거나 차폐할것
5) 활산되는 빛을 이용할것
6) 채광의 방향성을 고려할것
7) 환경적 요일을 고려한 통합적인 설계를 할 것 (인공조명과의 통합)

방위에 따른 자연채광의 특성
남쪽면 : 언제나 빛이 일광되게 유입됨, 알사 차폐장치가 효과적
북쪽면 : 빛의 양은 적지만 질이 높음, 글레어가 적음
동서쪽면 : 하루의 반만 빛을 받음, 여름에 빛을 최대로 받음, 고도가 낮아 조절하기 힘든 글레어 발생

형태별 채광효율
주광을 이용하기 위한 건물형태 : L형, U형, 도넛형, 계단형, 선형
건물의 두께를 줄이고 건물 전체에 충분한 주광을 삽입하는 것이 효율이 좋음

선형형태
기본적으로 건물의 향이 중요한 형태
건물의 길이가 동서향일 경우 계절적으로 자연형 난방 냉방과 부합
선형형태의 자연채광은 실 깊이와 관련됨

중심형태 : 내부코어를 갖는 형태, 가로와 세로의 비가 같음
군집형태 : 여러 다양한 형태의 일련의 작은 매스로 구성 다양한 각도에서 채광 가능


창 : 자연채광 도입을 위한 디자인 요소
칭의 기능 : 주광의 유입, 통풍, 환기, 조망감, 개방감, 단열, 방음, 방풍 등

창의 위치별 역할
높은 창 : 주광을 실내 깊숙이 사입, 균일한 배광, 시야차단, 천장과 상부벽은 반사율이 높은 무광의 벽이 좋음
중앙 창 : 바닥에 입사, 지면에 반사된 빛을 널리 배분
한쪽 끝창 : 조망에 제한, 실의 형태와 크기를 이해하는데 용이

실의 깊이는 효과적인 빛 배분을 위해 창의 높이(바닥~창의 꼭대기)에 2.5배를 초과하면 안됨

투과체의 성능
열관류값(U Value) : 열에너지가 물체에 의해 전달되는 정도를 수치화 한 것, 단열성능
일사획득계수(SHGC) : 창호를 투과하는 일사획득의 퍼센트 비율, 냉난방성능
음향투과손실(TL) : 음향 차음성능, 소음차단능력
가시광선투과율(VT) : 빛의 투과성능, 채광성능


채광방식
자연형 채광방식
-측창채광 : 저측창 채광, 측창 채광, 고측창 채광
-천창채광
-정층창채광

층창채광 : 측면에 창을 내어 채광하는 방식
한쪽에 창을 뚫는 편측채광, 양쪽에 창을 뚫는 양측채광이 있음
구조 및 시공, 조작이 용이하지만 조도 분포가 불균일하고 주변 건물에 채광이 방해받을 수 있음. 또한 글레어가 발생

층창채광 기법
1) 창호는 벽에 높은 곳에, 넓은 면적으로, 적당한 크기로 위치하여야 한다.
-창호가 여러개일 경우 주광은 더 균등하게 분포
-바닥면적 대비 창호면적은 20% 넘지 않게 -> 여름엔 과열 방지, 겨울엔 열손실 방지 목적
-반사판이나 환산판 등을 이용하여 주광 획득
2) 주광을 천장에 반사시켜 보다 깊은 공간에 균등히 사입되도록 한다.
-1층 건물일 경우 밝은 색의 안뜰, 보도 반사광을 이용
-다층 건물일 경우 구조의 일부가 빛을 반사하는데 이용
3) 가능한 한 창호는 1개 벽면 이상에 설치한다.
-가능한 편측채광보다 효율이 좋은 양측채광을 사용
4) 창턱이나 창 위의 반사판을 이용하여 빛을 실내부 깊은 곳에 반사시킨다.
5) 창을 실내벽 가까이 설치한다.
-창호 근처의 실내벽은 낮은 밝기의 반사체 역할을 해서 빛의 강한 방향성을 줄임, 과한 휘도비 감소
6) 직사일광은 차폐하고 주광은 최대한 사입시킨다.
-사무공간 내에는 직사일광 차단
-직사일광을 확산시키거나 천장에 반사시켜 사입
7) 주광을 여과한다.
-나무, 격자울타리 또는 스크린을 이용하여 여과된 빛을 유입
-넓은 면적의 반투명 유리나 밝은 색의 커튼으로 직접 현휘를 일으킬 수 있음

천창채광 : 지붕면에 대해 수평 또는 수평에 가까운 창에 의한 채광
벽과 바닥의 반사는 제 2의 광원으로 역할할 수 있음
시야차단 효과가 있어서 폐쇄된 분위기를 연출

광정 : 외부의 빛을 천장 아래로 확산시키기 위해 확산유리를 사용하여 빛의 우물같은 형태로 만든 것

천창채광 기법
-균일하고 높은 조도레벨을 가짐
-다층건물에 적용이 불가하고 조망과 방위조절에 대한 요구를 만족시키지 못함
-과할경우 글레어 발생, 광막반사의 원인이 됨
-빛을 천장으로 반사시키거나 차폐장치를 사용하여 광원을 가림
-반투명유리 사용 가능, 직접 현휘를 피할 수 있다는 장점이 존재
1) 균일한 조명을 위한 천창의 간격
-수직창호가 있는 경우에는 천창은 외주부에서 더 멀리 떨어져도 됨
2) 천창을 북쪽 벽 위에 설치한다.
-북쪽벽은 천창 빛의 확산체로 이용
-북쪽 천창은 남쪽의 수직 개구부로부터 조도와 균형을 이룰 수 있음
3) 여름과 겨울에 주광 사입의 균형을 개선하기 위해 가파르게 경사진 천창을 사용
4) 천창의 실제 크기를 증가시키기 위하여 벌어진 개구부를 사용한다.
-수평천창의 경우 : 여름철에 많은 빛과 열을 사입하므로 북쪽 또는 남쪽으로 배치
-경사진 천창의 경우 : 연중 균일한 빛을 사입
5) 여름과 겨울에 균형된 주광사입을 개선하기 위하여 챠양과 반사판 사용
-여름 빛 차폐, 겨울 빛 취득을 증가시키기 위해 반사판을 사용
6) 태양빛을 확산시키기 위하여 실내 반사판을 사용 (킴벨미술관)
-개구부 아래에 반사판을 설치하여 균질의 확산광 제공
7) 천창을 공간의 높은 위치에 설치
-빛이 바닥에 도달하기 전에 확산
-관측자의 시야가 모서리에 있어서 직접 현휘를 피할 수 있음
8) 천창을 통해 깊은 공간에 빛을 전달하기 위해 일광반사장치를 이용
-태양을 추적하여 태양의 각도와 상관없이 빛을 건물 하부로 사입
9) 정측창을 이용

정측창채광 : 넓은 작업면에 주광의 분포가 일정해야 하는 곳에 사용
천창채광과 측창채광 효과를 동시에 얻기위해 지붕면에 수직 형태의 창을 두어 채광 (공장 느낌)

정측창채광 기법
1) 모니터, 톱날형 창의 간격
2) 지붕의 반사율을 높게 한다.
-지붕 반사광은 낮은 밝기의 높은 질을 갖는 조명이며, 열취득을 감소시킴
3) 개구부를 남쪽을 향하게 한다.
-연중 매우 일정한 빛이 사입되게 하고, 겨울철에 태양열 난방의 효과
-직사일광 관련 문제들이 야기되지 않도록 개구부를 설계
4) 실내 조명의 조화를 위해 태양의 차폐장치를 설치한다.
-북향의 경우 외부에 반사된 빛을 유입하도록
-동서향의 정측창은 바람직하지 않으나 반사광유입을 하여 개선
-동향은 아침의 빛의 일부를 차폐하고 오후에는 반사된 빛의 양을 증가, 서향은 반대
5) 실내 벽면에서 빛을 반사시킨다.
-벽은 대형의 낮은 휘도를 가진 확산체를 사용
6) 높은 질의 조명을 위하여 차양이나 확산판을 이용 (몽트 에리 도서관)
-남향의 정측창은 차양에 의해 매우 확산, 눈부심이 감소, 높은 조도가 가능
